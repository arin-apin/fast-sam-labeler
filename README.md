# fast-sam-labeler

## Usage:
Set image_folder, output_folder, labels variables 

Keys:
 - a:      Previous image 
 - d       Next image 
 - 0-9     Label selection 
 - r/m Change mode (auto/manual) 
 - esc/q   Exit
 - +/-     Increase or decrease confidence for SAM auto mode

## Requeriments
Pytorch (gpu support optional)<br>
Ultralytics
